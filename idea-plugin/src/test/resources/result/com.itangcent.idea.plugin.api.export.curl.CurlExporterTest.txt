## say hello
```bash
curl -X GET http://localhost:8080/user/greeting
```

---

## get user info
```bash
curl -X GET http://localhost:8080/user/get/{id}?id=0
```

---

## create new user
```bash
curl -X POST -H 'Content-Type: application/json' -d '{
  "id": 0,
  "type": 0,
  "name": "",
  "age": 0,
  "sex": 0,
  "birthDay": {},
  "regtime": {}
}' http://localhost:8080/user/add
```

---

## update user info
```bash
curl -X PUT -H 'Content-Type: multipart/form-data' -F 'id=0' -F 'type=0' -F 'name=' -F 'age=0' -F 'sex=0' http://localhost:8080/user/update
```
