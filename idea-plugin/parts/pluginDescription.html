Help you export API to yapi/postman/markdown
  • Parse API documents based on javadoc & KDoc & ScalaDoc
  • Note that you can get a fairly complete API document in the case of zero intrusion of the code.
  • However, special requirements still need some special comments/annotations. Compared with this, you can use flexible configuration rules to adapt to the characteristics of your project to minimize code intrusion.

Refer to Guide for details
    Export API to Yapi/Postman/Markdown:

    Open the file containing api in the project or select the file or folder in the IDE's left project file area, use the shortcut Alt Shift E (windows) / Ctrl E (mac), Then select the API you want to export, select the export channel Yapi/Markdown/Postman, Click the [✔] button or press Enter to complete the export.


If the shortcut is invalid, please check for shortcut conflicts. You can modify the shortcut in Preferences(Settings) > KeyMap.
