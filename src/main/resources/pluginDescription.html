EasyYapi


GitHub | Issues | Guide

Export API to Yapi / Postman / Markdown / Curl / HttpClient
  • Parse API documents from Javadoc, KDoc & ScalaDoc
  • Support Spring MVC, Feign, JAX-RS (Quarkus, Jersey)
  • Support javax.validation, Jackson, Gson, Swagger annotations
  • Flexible configuration rules to minimize code intrusion

How to Use:
  1. Right-click on a controller file, class, or method in the editor or project view
  2. Select Export API from the context menu
  3. Choose the target channel (Yapi / Postman / Markdown / Curl / HttpClient)
  4. The APIs will be exported automatically

For more details, refer to the Guide
